# Stripe Payment Link の作り方

## 作成する商品（MVP）

コンサルタント3人 × 料金プランで作成。シンプルに30分1種類から始める。

| 商品名 | 金額 | 備考 |
|--------|------|------|
| コンサルティング（山田）30分 | ¥5,000 | コンサルタントごとに作成 |
| コンサルティング（佐藤）30分 | ¥5,000 | 料金が同じでも別商品にする |
| コンサルティング（鈴木）30分 | ¥8,000 | 料金が違う場合 |

**コンサルタントごとに商品を分ける理由**: 売上管理・分析がしやすい

---

## 作成手順

### 1. Stripeダッシュボードにログイン

<https://dashboard.stripe.com>

---

### 2. 商品を作成

```
左メニュー「商品カタログ」→「商品を追加」
```

入力項目：

| 項目 | 入力例 |
|------|--------|
| 名前 | コンサルティング（山田太郎）30分 |
| 説明 | 任意（空でもOK） |
| 画像 | 任意（コンサルタントの顔写真など） |
| 価格 | ¥5,000 / 1回（One time） |

**注意**: 「継続」ではなく「1回限り（One time）」を選択

---

### 3. Payment Link を作成

商品作成後、そのまま：

```
「Payment Linkを作成」ボタンをクリック
```

または：

```
左メニュー「Payment Links」→「新規」→ 商品を選択
```

---

### 4. Payment Link の設定

| 項目 | 設定 |
|------|------|
| 商品 | 作成した商品を選択 |
| 数量調整 | オフ（1回固定） |
| 支払い後のリダイレクト | `https://moment-works.com/bookings/confirmed` |
| 顧客情報の収集 | メールアドレス（必須）、名前（任意） |

---

### 5. リンクをコピー

作成完了すると、こんなURLが発行される：

```
https://buy.stripe.com/xxxxxxxxxxxxxx
```

これをSupabaseの `consultants.payment_link` に保存。

---

## 使い方（運用時）

コンサルタントがクライアントに送るとき：

```
https://buy.stripe.com/xxxxxxxxxxxxxx?client_reference_id=booking_abc123
                                      ↑
                               予約IDを付与（自動紐付け用）
```

---

## テスト方法

### テストモードで作成

Stripeダッシュボード右上のトグルで「テストモード」に切り替えてから同じ手順で作成。

テスト用カード番号：

```
4242 4242 4242 4242
有効期限: 未来の日付
CVC: 任意の3桁
```
